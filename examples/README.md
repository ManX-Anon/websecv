# Examples

This directory contains example code for using the scanner.

## Examples

- `example_plugin.py`: Example plugin implementation
- `example_scan.py`: Basic scanning example
- `example_intruder.py`: Fuzzing example
- `example_crawler.py`: Web crawling example

## Running Examples

```bash
# Example plugin
python examples/example_plugin.py

# Example scan
python examples/example_scan.py https://example.com
```
